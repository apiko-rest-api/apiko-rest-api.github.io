Apiko - The easiest way to create a REST API

OK well. How long does it take to learn?

Wrong question. You can already create projects with Apiko.

All you need to start creating is to do:

npm i -g apiko-cli
apiko setup myapikoserver

Apiko will take a while to install everything in the 'myapikoserver' directory and then opens in a browser tab. If it doesn't, go to http://localhost:5000/dev/.
